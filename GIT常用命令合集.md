# GIT常用命令合集

| 功能                       | 指令                       |
| :------------------------- | -------------------------- |
| 新建仓库                   | git init                   |
| 添加文件到暂存区           | git add  文件名            |
| 将文件从暂存区提交到仓库   | git commit -m '提交的注释' |
| 显示工作目录和暂存区的状态 | git status                 |
| 查看修改内容               | git diff 文件名            |
